## Contribution

- Clean code (pep-8)
- Documentation
- Tests

## Development environment
- clone repo
- `poetry install`
- `poetry shell` 

## Running tests
- `make tests`

## Generating docs
- `make docs`


## building dists
- `poetry build`

## publishing 
- `poetry publish`
